Api to count words from a given url


## Setup

```bash
virtualenv --python=/usr/local/bin/python3 env

source env/bin/activate

pip install -r requirements.txt
```

## Runing Server

```bash
run.sh
```

## Runing tests

```bash
pytest .
```
